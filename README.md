# Egg-Incubator-Automation
**الوصف:**   نظام ذكي لفقس البيض بسعة 60 بيضة، يتحكم أوتوماتيكياً في درجة الحرارة والرطوبة ويقلب البيض لضمان نجاح الفقس.
# Solar Pipe Cleaning System

**الوصف:**  
آلية أوتوماتيكية لتنظيف الأنابيب الشمسية المفرغة، مع مفاتيح نهاية للتحكم في حركة النظام وحماية المعدات.

**الأدوات والمكونات المستخدمة:**  
- Arduino Mega  
- محركي خطوة (Stepper Motors)  
- حساس غبار GP2Y1010AU0F  
- درايفر TB6600  
- أسلاك ومكونات إلكترونية داعمة  

**الدور والمسؤوليات:**  
- تصميم النظام الإلكتروني بالكامل  
- برمجة حركة النظام والتحكم بالأجهزة والمفاتيح  
- اختبار النظام وضبط الحركة ليتم تنظيف الأنابيب بدقة  

**النتائج:**  
- تنظيف فعال للأنابيب الشمسية مع تقليل الوقت والجهد البشري  
- نظام آمن وموثوق للاستخدام الطويل
